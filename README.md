# UHear
A synchronized music player based on Flutter

注意，要自己运行需要将Flutter中的api接口改成自己的，其中网易云api使用的是https://github.com/fluttercandies/NeteaseCloudMusic

已完成：

- 从网易云接口搜索并播放音乐
- 创建随机房间号与用户id
- 加入房间
- 播放、暂停、停止时在房间内同步操作
